Github

GitHub is a developer platform that allows developers to create, store, manage and share their code. It uses Git software, providing the distributed version control of Git plus access control.

Stacflow

Stack Overflow is a question-and-answer website for computer programmers. It is the flagship site of the Stack Exchange Network.It was created in 2008 by Jeff Atwood and Joel Spolsky.

Reddit

Reddit is an American social news aggregation, content rating, and forum social network. Registered users (commonly referred to as "Redditors") submit content to the site such as links, text posts, images, and videos.
